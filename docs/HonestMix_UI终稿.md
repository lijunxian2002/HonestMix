## 三、色彩系统

### 统一色谱（推子 0→200）

| 位置 | 颜色 | 色值 |
|:--|:--|:--|
| 0 (红) | 鲜红 | `rgb(255,50,35)` |
| 25% | 橙 | `rgb(255,90,30)` |
| 50% | 金 | `rgb(255,150,40)` |
| 68% | 黄绿 | `rgb(240,195,50)` |
| 82% | 翠绿 | `rgb(150,210,55)` |
| 92% | 青绿 | `rgb(50,210,80)` |
| 100% (绿) | 鲜绿 | `rgb(20,220,110)` |

**所有颜色组件统一调用 `spectrumColor(pct)` 函数**：
- VU 条填充色
- 曲线面板边框
- CORR 曲线描边 + 填充
- 状态灯

## 四、交互逻辑

### 推子
- 上下拖拽 / 滚轮 → 值 0↔200
- 值变化 → VU/边框/CORR 线/CORR 底/状态灯 统一变色
- CORR 曲线形状实时插值（0=RAW 形状 → 200=平直线）

### 曲线面板
- **默认**：仅显示 CORR 线（颜色跟随推子）
- **鼠标悬停**：双线黄金比例（红 0.618 / 绿 0.382）预览对比
- **点击**：切换 RAW 线显隐
- **悬停曲线**：tooltip 显示 Hz + dB

### 下拉选择器
- **Headphone**：自定义搜索面板（品牌分组，输入过滤，~50 示例）
- **Target Curve / Interface**：原生 `<select>` 下拉

## 六、已知限制（留给 JUCE 开发者）

1. **Headphone 下拉**：HTML 用自定义 JS 实现。JUCE 需用 `TextEditor` + `ListBox` 接入 `HeadphoneDatabase`（~1600 条）
2. **颜色系统**：`spectrumColor()` 函数需移植到 `Design.h` 作为 `hm::spectrumColor(float pct)`
3. **曲线插值**：`rawYs[]` 数组为示例数据，需替换为实时耳机曲线采样点
4. **NASA 背景**：`stars_deep.jpg` 仅供原型展示，正式版可移除或替换为纯色
5. **推子材质**：CSS 渐变模拟金属，JUCE 中可用 `ColourGradient` 实现
