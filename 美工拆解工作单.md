# HonestMix · 美工拆解工作单

> **你的任务**：把 `HonestMix_终版.html` 拆成 13 个独立组件文件，一次只做一个。
>
> **你的参考**：`design-tokens.md`（所有色值、字体、间距、圆角、阴影、动画参数都从里面取）。
>
> **你的模板**：每个组件用下面这份 HTML 骨架，填肉就行。
>
> **核心规则**：CSS 里禁止硬编码色值。全部用 `var(--xxx)` 引用令牌。

---

## 通用组件模板

每个组件文件都长这样：

```html
<!-- ============================================================
  组件：XX · 组件名
  设计令牌参考：design-tokens.md §X
  拆解日期：
  状态：□ 草稿 □ 待审 □ 定稿
  ============================================================ -->

<!DOCTYPE html>
<html lang="zh-CN">
<head>
<meta charset="UTF-8">
<title>HonestMix · XX · 组件名</title>
<style>
/* ===== 本组件用到的令牌（从 design-tokens.md 复制） ===== */
:root {
  /* 背景色 */
  /* 文字色 */
  /* 边框色 */
  /* 间距 */
  /* 圆角 */
  /* 阴影 */
  /* 字体 */
  /* 动画 */
}

/* ===== 组件样式 ===== */
/* 只写 var(--xxx)，不写硬编码色值 */

/* ===== 状态变体 ===== */
/* 每个状态一个 CSS class */
</style>
</head>
<body style="background:#1a1a18; display:flex; align-items:center; justify-content:center; min-height:100vh; padding:40px;">

  <!-- ===== 默认态 ===== -->
  <div class="xxx" data-component="XXX" data-state="default">
    ...
  </div>

  <!-- ===== Hover 态 ===== -->
  <div class="xxx hover" data-component="XXX" data-state="hover">
    ...
  </div>

  <!-- ===== Active 态 ===== -->
  <div class="xxx active" data-component="XXX" data-state="active">
    ...
  </div>

  <!-- （有多少状态就写多少个 DOM 块） -->

</body>
</html>
```

## 常见问题

**Q: 令牌名我忘了去哪查？**
A: `design-tokens.md` 目录里搜关键词，比如想知道 Chip 边框色 → 搜 "border-chip"。

**Q: 有些令牌在 design-tokens.md 里没有怎么办？**
A: 先来问我。如果确实缺失，我们两个人一起决定色值，然后我同步更新 design-tokens.md。

**Q: 状态太多了做不完怎么办？**
A: 先做 default + 1 个最重要的交互态，其余标注 "工程师参考 §X 自行扩展"。

**Q: 动画怎么办？CSS 动画还是 JS？**
A: 简单的（hover 变色、fadeIn）用 CSS transition。复杂的（星空呼吸、心形编队）用 JS 驱动。把动画参数标注清楚就行。
